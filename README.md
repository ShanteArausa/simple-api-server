# Simple API Server

A basic Node.js + Express backend API built as part of my backend fundamentals training.

## Features
- GET /
- GET /users
- GET /products
- POST /users

## Tech Used
- Node.js
- Express

## How to Run
1. npm install
2. node server.js
3. Visit http://localhost:3000

## Purpose
This project demonstrates:
- Backend routing
- Handling GET & POST requests
- JSON responses
- In-memory data storage
